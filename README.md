# Simple Library

This is a small  laravel project with restful api


## Run App

$ git clone <br />
$ cp .env.example .env <br />
$ composer install <br />
$ php artisan migrate <br />
$ php artisan db:seed <br />
$ php artisan serve <br />

## Run Tests
 
$ phpunit <br />
